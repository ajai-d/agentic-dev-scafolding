# Supply Chain POC

Baseline scaffold for a domain-separated Supply Chain proof of concept.

## Domains
- Order Management
- Inventory Management

## Stack
- Frontend: React + Vite
- Backend: Node.js + Express
- Database target: SQLite (implementation in later execute steps)

## Run Locally

From project root:

```bash
npm install
npm run dev
```

Default ports:
- Frontend: http://localhost:5173
- Backend: http://localhost:3002

## Baseline Endpoints
- `GET /health`
- `GET /api/info`

## Notes
This is Execute E1 setup baseline only. Domain features are planned for subsequent execute units.
